# Project-B
A site to track and display tournament stats and data

Built with: React.js Node.js Express and MongoDB

featuring: react hooks, Context api, axios, react bootstrap, json web tokens, bcrypt

## Dependencies:
 ### Backend
 express
 mongoose
 dotenv
 cors
 bcrypt
 jsonwebtoken
 
 ### Client
 axios
 bootstrap
 react-router-dom
 react-icons
 react-scripts
